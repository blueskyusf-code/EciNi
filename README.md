# EciNi
EciNi é um projeto Código Aberto, você pode fazer o que quiser nele, fazer jogos, apps etc, tudo unificado em um só lugar, apenas use Java e faça o que quiser com ele
<br>
<br>
O Projeto por padrão vem com um arquivo java como menu, você pode modificar ele pra adicionar suas funções como jogos que tú quiser ou apps
